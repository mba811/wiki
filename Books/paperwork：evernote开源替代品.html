paperwork：evernote开源替代品

习惯了 EverNote 却对其收费及云端访问不放心？Paperwork 是一个可以架设到自己服务器上的开源替代品。

Paperwork 具备以下特点：

  • 使用 Laravel 4 PHP 框架开发，提供可供第三方集成的 API

  • 可运行于具备 LAMP 的服务器或者 Synology 及 QNAP NAS 中

  • 易于通过 Composer 安装及 Docker 部署

  • 具备历史记录、文件上传等功能

  • 基于 CKEditor 的富文本编辑器

  • 依据 MIT 许可分发

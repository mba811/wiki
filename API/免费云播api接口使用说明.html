免费云播API接口使用说明

大云播提供了一个简单方便的云播接口和云播调用方法，您只需提供ed2k、magnet磁力链等视频链接地址就可以使用。

接口地址 ：
http://www.dayunbo.com/api/api.php#!u=

url后面加上播放的地址即可，版面将随浏览器窗口大小变化而变化的，具体可由你自己决定。

云播放接口页面完全静态化，用javascript都可以完全控制，适用于所有web服务器环境，当然包括PHP，ASP，或是.NET。

1.直接调用或软件调用方式

http://www.dayunbo.com/api/api.php#!u= 后面加上播放的地址即可

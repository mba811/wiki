JSON简历

它是什么？

JSON简历是一个社区驱动的开源倡议建立基于JSON的标准简历。

为什么JSON？

我们认为，JSON格式的优势使得它非常适合简历。它是轻量级的，易于使用，它是完美打造的工具！

我们还认为，JSON模式是用于编写可用语义不够成熟。

集成

导出您的个人资料从下方服务的JSON格式的简历：

Represent.io

http://represent.io

代表的是要在几分钟内创建一个美丽而专业的简历的最佳途径。

DoYouBuzz

http://doyoubuzz.com/

创建简历从未如此简单。在几分钟内，你将有一个漂亮的简历，你可以在任何地方使用。

命令行工具

我们已经建立了一个CLI（命令行界面），它支持OSX，Linux和Windows。要创建自己的简历，安装resume-cli从NPM：

sudo npm install -g resume-cli

出口

命令行工具使用，我们已经开源了你的简历转换成不同的格式的模块组成的生态系统：

resume export resume.pdf

resume export resume.html

出版

我们还建立在支持托管到命令行工具。就像故宫的注册表，你可以注册一个帐号，并发布到我们的服务器。我们也可以让您的密码保护和/或匿名发布你的简历。上手，只需键入：

resume publish

我们的托管解决方案，将有新的功能并陆续推出相当频繁的好处。请确保您注册，如果你想保存的虚荣心网址：

resume register

太好了！你现在应该有自己的网址，如：http://registry.jsonresume.org/thomasdavis

您也可以访问导出版本的简历：http://registry.jsonresume.org/thomasdavis.pdfhttp://registry.jsonresume.org/thomasdavis.txt

用户还可以使用密码保护他们的简历：http://registry.jsonresume.org/richardhendriks

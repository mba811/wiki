Docker-Hub使用

开始使用Docker Hub

Docker Hub是docker的中心仓库。它里边的公共docker镜像并且帮助您构建和管理docker镜像。了解更多。阅读更多Docker Hub

  1. 展示如何创建一个账户。

  2. Docker Hub和docker及其组件一起工作。Docker Hub会帮助你和你的同事协作，并获取功能最全的docker。要做到这一点，它提供的服务有：

    1. Docker镜像主机

    2. 用户认证

    3. 自动镜像构建和工作流程工具，如构建触发器和webhook

    4. 整合了GitHub和BitBucket

  3. 为了使用Docker Hub,首先需要注册创建一个账户。别担心，创建一个账户很简单并且是免费的。

创建Docker Hub账户

  1. 这里有两种访问可以创建和注册一个Docker Hub账户：

    1. 通过网站创建

    2. 填写注册表单,选择您的用户名和密码并指定您的电子邮箱。你也可以报名参加docker邮件列表，会有很多关于docker的信息。

    3. 一旦你填写表格，查看你的电子邮件，欢迎信息，确认来激活您的账户。

    4. 在完成确认过程之后，您可以使用web控制台登陆

    5. 通过命令行创建

    6. 你可以创建一个Docker Hub账号通过命令行docker login命令

    7. $ sudo docker login

你的Docker账户现在已经激活，你可以使用了。

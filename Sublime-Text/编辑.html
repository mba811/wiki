编辑

概观

崇高文字边缘，充满了编辑功能。本主题只是触及了什么是可能的表面。

列选择

列选择可以用来选择文件的一个矩形区域。列选择不通过单独的模式下运行，而不是它利用多个选择。

您可以使用添加剂的选择，选择文本的多个块，或消减选择删除块。

使用鼠标

视窗

  • 鼠标右键+ ⇧

  • OR：鼠标中键

  • 添加到选区：按Ctrl

  • 从选区中减去：Alt键

Linux的

  • 鼠标右键+ ⇧

  • 添加到选区：按Ctrl

  • 从选区中减去：Alt键

OS X

  • 鼠标左键+ ⌥

  • OR：鼠标中键

  • 添加到选区：⌘

  • 从选区中减去：⌘ + ⇧

使用键盘

窗口：按Ctrl + Alt键 + 向上和按Ctrl + Alt键 + 向下

Linux的：Alt键 + ⇧ + 向上和Alt键 + ⇧ + 向下

OS X：^ + ⇧ + 向上和^ + ⇧ + 向下

多重选择

多种选择让你进行全面调整，以有效文本。关于多个选择任何赞美是轻描淡写。这就是为什么：

选择一些文本，然后按Ctrl键 + ð来添加更多的实例。如果你想跳过当前实例，按按Ctrl + K， Ctrl键 + ð。

如果你走的太远，按按Ctrl + ü要取消当前实例。

转化多重选择到线

CTRL + L扩大选择到行的结尾。按Ctrl + Shift键 + L 分割的选择进行。

您可以选择多行复制到一个单独的缓冲区，编辑那里，再次选择内容为多行，然后将它们粘贴回原处的第一个缓冲区。

选择文本的其他方式

名单很长; 所有可用的选项下可以找到的选择。仅举几例：

  • 选择子字（Alt键 + Shift键 + <箭头>）
  • 扩展选择，以括号（按Ctrl + Shift键 + M）
  • 扩大选择压痕（按Ctrl + Shift键 + Ĵ）
  • 扩大选择范围，以（按Ctrl + Shift键 + 空格）

移调事情

需要交换两个字母，或者更好的是，这两个词？实验 按Ctrl + T。

Marked 2 使用方法集锦

首先妳得会用 Markdown 语言编辑文本，MD 很容易掌握，你可以先看看创始人 John Gruber 写的语法，然后再看看我们站内推荐的各种 MD 软件，基本上就算入门了，而 Brett 这款 Marked 2 也算把 Markdown 玩到了前所未有的境界，这是一款功能强大的 Markdown 标记文本预览软件，能辅导你撰写出最专业的 MD 文本，但我一直纳闷，为啥 Brett 不自己再开发一款 MD 编辑软件呢？也许是害怕与罗晨竞争吧？哈哈，开个玩笑，回归主题。

经过了一年半的锤炼，第二代的功能全面性已今非昔比，针对不同使用人群，Marked 都推出了相应的实用功能，在今晚构思文章架构的时候，觉得一个晚上的时间还是不能道出所有的功能，所以今天这篇就算个引子吧，可能对 Marked 2 持观望的朋友会失望了，不能在今晚看到完整的评测，但我只能说，如果你喜欢撰写有条理的干净文章，那 Marked 2 绝对是最佳的工作伴侣，先来举个例子好了：无论你是使用 Ulysses、Mou、Byword 这样的轻量化编辑工具，还是使用 Scrivener 写大布头长篇， Marked 都能流畅的加载文本进行预览，Marked 根据 MD 文章内子标题对全文段落进行自动识别，并给出文章索引目录方便作者审阅，把握行文节奏，支持许多诸如 MultiMarkdown，Fountain 等泛标记文本语言，你可以随时查看文章的字数，段落数，预计阅读所用时间，并验证文内所有外链的有效性等等功能，就像是一个复杂的检测机器，能够发现并纠正你的文章里每一处编辑错误。

在 Marked 预览窗口标题栏上，按住 CMD 键，单击标题还能弹出 Finder 目录选择菜单，选择任意一个目录都能弹出文件选择窗口：

当然，Marked 2 打开文本的普通方式也不缺少，像在 Menubar 上选择「Open」，「Open Recent」都算最基本的了：

除此之外，Marked 还能直接从剪切版文本记录里直接预览效果，在 Menubar 菜单里进入 Preview – Clipboard Preview 可以直接生成预览窗口，或者使用热键 「⌘⇧V」也可。

阅读控制组功能「核心功能」

1. Table 「文章导航」

Marked 核心功能之一就是：Table，即文章内容导航，如果你的文章带有 H1，H2 这样的子标题，那么 Marked 会识别文内的各级标题，自动生成，点击底部右侧的工具组第二个图标会弹出文章导航，也可按「F」键弹出，但这样也会触发段落快速跳转功能，按「F」键后在文章导航里输入分段落子标题并按回车就能快速跳转到分段落。

使用「J」和「K」键或着上下方向键能在文章导航里的各个子标题之间移动。

2. Auto-scroll 「自动滚动翻页」

在浏览长文时你需要自动滚动翻页这样的功能来解放你的手指去不停的按翻页键，按「S」键开始自动滚动，按住「Shift」键，再按左右方向键可以逐步加快或减慢滚动速度。

3. 检查外链

Marked 可以对文内所有外链进行检查，鼠标单击不要马上放开，过1秒松开会弹出工具菜单，你可以选择验证外链是否合法，将外链地址拷贝到剪切板，或是选择在 Safari 里打开：

4. 搜索

Marked 2 的文本搜索功能不是非常丰富，除了提供精准词汇，大小写区分以及正则表达式这三种筛选条件外再没有其他功能了：

5. 阅读进度动态指示条

当你在阅读长文的时候，你会发现顶部标题栏实时变动的蓝色进度条会明确告诉你当前阅读的位置：

6. 段落折叠

段落折叠功能同样是为多段落的长篇文本所设计的功能，按住 CMD 键（或在设置里选择不按住），单击分段标题就能对这段文本进行折叠，当你按住「Shift」再单击标题时还能快速将其他同级的分段落折叠。使用热键 「Option」+「Command」+ 左右方向键还能快速将全文文本折叠或铺展。

7. Mini Map

Marked 2 真是为长文所设计的，按数字「0」键可在预览窗口右侧生成一个用来快速浏览正文的缩小版全文截图，即 Mini Map，当然，这可是活的 Map，当你按住 CMD 键将光标放在里面上下移动时，会弹出一个当前段落的预览窗口，不过我感觉 Mini Map 这个功能不如文章导航「Table」来的实在，使用频率不会太高，而且每次开启 Mini Map 时会造成软件卡顿。

8. View Source

使用热键「CMD+U」或是窗口左下角的「<>」按钮，可以在预览模式与 HTML 源码模式下快速切换

导航用快捷键

Marked 2 的热键无比丰富，可以说每一个小功能都有热键来配合实现，在你需要回顾某些热键时，可以按「h」直接在窗口左侧调出常用热键列表，使用组合键「CMD+Option+K」可以调出一个更完整的热键参考悬浮框：

「J」和「K」：上下滚动翻页

「Shift」+「J/K」：加快滚动速度

「T」：一键回到文章顶部

「B」：一键回到文章底部

「U」：以当前窗口尺寸二分之一的距离向后滚动

「D」： 以当前窗口尺寸二分之一的距离向前滚动

「英文逗号/句号」：在各个同级子标题所在分段落快速跳转

Bookmarks「给文章添加阅读书签」

阅读时插入书签这个动作本来是真实生活中一个常用方法，在虚拟屏幕上她也会经常被用到，Marked 2 设计的书签功能非常好用，当你遇到需要二次修改或反复雕琢的段落时，可以按住「Shift」+「1-9任意数字」可以为该段落添加标签，添加后你会在段落右上角看到标签数字，按对应数字就能瞬间跳到那个段落上。

使用「n」，「p」两个单键可以上下顺序跳转段落，如果添加标签时，把「Shift」换成「Option」按住再选择数字可以对子标题看成标签添加；此外还有一个小细节需要注意，按数字「0」键可以将所有书签的数字改为分段落子标题显示，这样将更为直观，双击「x」键将清除所有已添加的书签

导出

1. 基础导出功能

在编辑出满意的作品后，格式输出也是最后一个值得注意的环节，每个人的需求都不一样， Marked 2 也是尽可能的照顾到了所有人的需要，她能将 MD 文本导出成： HTML, PDF, RTFD, RTF, DOCX, ODT 等多种格式的文档，如果你只是想立刻将文本拷贝到浏览器的编辑器里，使用组合键「CMD+Shift+C」可以一键拷贝全文HTML源码到系统剪切板里，看下面的动图：

2. PDF 打印

有许多用户需要把 MD 文章输出成 PDF 格式并打印出来的，Marked 2 提供的 PDF 打印功能可谓十分给力，你可以将各项打印参数调整好后保存为一个「Preset」，每次调用时直接在“Presets”选项里勾选即可：

如果你想更进一步设计自己的打印排版，可以在软件设置-Export里对分页方式，页间距，表格内容的打印（分级标签规则），页头，页尾的内容定义，字体等参数进行规范：

比如拿页头，页尾来说，这些显示项可以放在页面四个边角位置，显示的信息可以是文章标题，存储路径，日期，时间等等，以下是官方给出的信息解释：

  • %title = Document Title
  • %date = Current Date
  • %time = Current Time
  • %page = Current Page Number
  • %total = Total Page Number
  • %path = Filesystem path to document
  • %%(text) = Text to print only on the first page

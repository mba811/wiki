小巧实用的文件加密工具：Encrypt

密码，在电子设备中随处可见，无论它用于何处、何种目的，都是为了避免不恰当的人在不恰当的时间看到不恰当的东西，所以它在我们生活中是不可或缺的。如果你是位 Mac 用户并且正在寻找一款小巧方便的文件加密工具，那不妨先试试免费的 Encrypto 。

若你有注意软件开发商的话，就会看到 MacPaw 的字样。MacPaw 家开发的软件实惠又好用，比如各位熟知的CleanMyMac便是出自他们之手，这次 Encrypto 更是以免费形式发布，值得一试。

作为一款纯粹的轻量级工具，操作便利性是很重要的，好在 Encrypto 对于这方面的把控非常合理，因此不会给我一种很累赘的感觉。具体是这样的：我们可以先打开 Encrypto 的主窗口，将需要加密的文件拖拽进去，进入接下来的设置页面；也可以直接拖拽至 Dock 栏的图标上实现。这是 OS X 的系统特性。

选定文件后，我们需要给它生成一个独立的密码提供保护，任何格式皆可。如果密码本身很难记忆，请设置文字提示。当然，你大可不必担心该密码的安全性，因为这是通过 AES-256 技术实现的，很可靠！确认无误后，经过一段 Encropt （加密） 动态效果的洗礼，一个被安全加密的文件就生成了。若配合Hider 2文件隐藏工具使用，保护性能会更好。

目前为止，加密后的文件必须通过 Encrypto 方可打开，这是硬性规定，不过好在这款应用同时支持了 Mac OS X 和 Windows 操作系统，因此用户在这两个平台间传输、读取加密文件是无障碍的。当然，Windows 版本也是免费提供下载的。另外，Encrypto 对被加密文件的大小没有要求，甚至于它在加密文件的同时，还会起到容量压缩的作用，读者们不妨尝试一下。

Encrypto 目前可 在 Mac App Store 下载 到 Mac 版，在官方网站 下载到 Windows 版 。

最近我发现，一物一用反而能将性能发挥得更彻底，轻量小巧的工具不仅功能纯粹，还可以压低制作成本，像这款免费的 Encrypto 便是如此。至于那些「全能」的软件，大部分人并不会用到全部功能，况且这类软件通常价格昂贵，实用性并不如意。

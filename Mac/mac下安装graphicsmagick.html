Mac下安装GraphicsMagick

下载依赖

GraphicsMagic根据使用功能的不同，依赖不同的组件，我需要的只是对jpeg和png进行简单的图片处理（缩放、裁剪啊什么的，不过它的功能可以更加强大），主要依赖了两个组件:

libjpeg

libjpeg-6b.tar.gz

libpng

libpng-1.4.7.tar.gz

GraphicsMagic

GraphicsMagic
GraphicsMagic 官网

安装

解压

进入下载目录解压。例如我下载的是libpng-1.6.17.tar.xz 就是下面的方法解压

tar jxvf libpng-1.6.17.tar.xz

运行

进入解压的目录，运行三条命令，然后就安装好了。

  1. ./configure
  2. make
  3. sudo make install

然后

然后按照上面步骤安装 GraphicsMagic
